# Social by Polybase

A simple social media demo for [Polybase](https://polybase.xyz) (the decentralized database).

🐦 [Twitter](https://twitter.com/polybase_xyz)



### Get Started

Load the database schema into Polybase:

```
yarn env yarn load
```

Then start the application:

```
yarn install
yarn start
```
